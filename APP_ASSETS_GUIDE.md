## 📱 APP ICON REQUIREMENTS

### Google Play Store Icon
**Size:** 512x512 pixels
**Format:** PNG (32-bit)
**Must be:** SFW (Safe For Work)

### Apple App Store Icon
**Size:** 1024x1024 pixels
**Format:** PNG
**Must be:** SFW, no transparency, no rounded corners

### ⚠️ WHAT NOT TO DO:
❌ NO anime girls (too sexy, will get rejected)
❌ NO explicit imagery
❌ NO hentai references
❌ NO "18+" text on icon
❌ NO text-heavy designs

### ✅ WHAT TO DO:
✅ Abstract logo/symbol
✅ Bold colors (purple/cyan gradients)
✅ Simple, recognizable design
✅ Professional looking
✅ Represents "creator platform"

### Icon Ideas for You:
1. **Stylized "FTW" letters** with gradient
2. **Camera + Paintbrush icon** (creator tools)
3. **Abstract waveform** (represents content/creativity)
4. **Minimalist planet/globe** (global platform)
5. **Geometric pattern** in your brand colors

### Quick Icon Creation:
**Option A: Use Figma (Free)**
1. Go to figma.com
2. Create 512x512 frame
3. Design simple logo
4. Export as PNG

**Option B: Use Canva (Free)**
1. Go to canva.com
2. Search "App Icon"
3. Customize template
4. Download as PNG

**Option C: Hire on Fiverr**
- Cost: $5-20

## 📸 SCREENSHOT REQUIREMENTS

### Google Play Store
**Phone screenshots:**
- Size: 1080x1920 to 1080x2400 (9:16 ratio)
- Format: PNG or JPG
- Minimum: 2 screenshots
- Recommended: 5-8 screenshots
- **ALL MUST BE SFW!**

**Tablet screenshots (optional):**
- Size: 1200x1920 or higher
- Format: PNG or JPG

### Apple App Store
**iPhone screenshots:**
- 6.7" Display: 1290x2796
- 6.5" Display: 1242x2688
- 5.5" Display: 1242x2208
- Minimum: 1 per device size
- Recommended: 3-5 screenshots
- **ALL MUST BE SFW!**

**iPad screenshots (optional):**
- 12.9" Display: 2048x2732

## 🎯 SCREENSHOT CREATION GUIDE

### Method 1: Take Real Screenshots
1. Run your app: `npm run dev`
2. Open in browser
3. Open DevTools (F12)
4. Toggle device toolbar (Ctrl+Shift+M)
5. Select device: Pixel 5 (1080x2340)
6. Navigate to different screens
7. Take screenshots (Ctrl+Shift+S or screenshot tool)
8. Save as PNG

### Method 2: Use Mockup Tools
**Mockuphone.com:**
1. Upload your screenshots
2. Select device frame
3. Download with device mockup
4. Looks professional!

**Figma/Canva:**
1. Create phone frame
2. Insert screenshots
3. Add text overlays
4. Export as PNG

## 💡 PRO TIPS

1. **Keep it simple** - Less is more for icons
2. **Test at small sizes** - Icons should be clear at 48x48
3. **Use mockups** - Device frames look professional
4. **Add captions** - Tell users what they're seeing
5. **Show progression** - Screenshots should tell a story
6. **Avoid clutter** - Clean UI screenshots only
7. **Test on phone** - How does it look on real device?
