# Fortheweebs

Fortheweebs is a sovereign creator-first platform built on top of Vanguard, designed for instant media deployment, legacy artifact creation, and mythic profile generation.

## Features

- Modular post creation and infinite scroll
- Tag filtering and mobile feed optimization
- Profile tribute logic for creator legacy
- CI/CD pipeline via GitHub Actions
- Automated moderation and backend sync

## Getting Started

```bash
npm install
npm run dev
```
