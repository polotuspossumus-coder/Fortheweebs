## 🎓 HOW TO USE THIS PLATFORM

### For Development
```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Analyze bundle size
npm run analyze

# Run tests
npm test

# Run linter
npm run lint
```

```bash
# Push to GitHub (auto-deploys to Netlify)

# Manual Netlify deploy
netlify deploy --prod
```
