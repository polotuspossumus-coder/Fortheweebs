## PHASE 4: BUILD APPS (Day 1-2)

### Android Build (1 hour)
- [ ] Run: `npm run build`
- [ ] Run: `npx cap sync android`
- [ ] Run: `npx cap open android`
- [ ] In Android Studio:
  - [ ] Build > Generate Signed Bundle/APK
  - [ ] Choose "Android App Bundle" (AAB)
  - [ ] Create keystore (SAVE THIS!)
  - [ ] Build release
  - [ ] Find AAB at: `android/app/build/outputs/bundle/release/`
- [ ] **CRITICAL:** Backup keystore file!

### iOS Build (Need Mac - 1-2 hours)
- [ ] Run: `npm run build`
- [ ] Run: `npx cap sync ios`
- [ ] Run: `npx cap open ios`
- [ ] In Xcode:
  - [ ] Sign with Apple Developer account
  - [ ] Product > Archive
  - [ ] Distribute > App Store Connect
  - [ ] Upload
- [ ] ✅ Build complete!
