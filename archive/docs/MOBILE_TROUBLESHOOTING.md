# 📱 Mobile App Troubleshooting Guide

## 🚨 Common Mobile Issues & Fixes

### Issue 1: App Won't Open / Crashes on Launch

**Fixes:**
1. **Force close and reopen**
   - iOS: Swipe up from bottom, swipe app away
   - Android: Settings → Apps → ForTheWeebs → Force Stop

2. **Clear app cache**
   - iOS: Delete and reinstall app
   - Android: Settings → Apps → ForTheWeebs → Clear Cache

3. **Update to latest version**
   - Check App Store / Google Play for updates

4. **Restart your phone**

---

### Issue 2: Can't Login / Stuck on Login Screen

**Fixes:**
1. **Check internet connection** (WiFi or mobile data)

2. **Clear app data**
   - iOS: Reinstall app
   - Android: Settings → Apps → ForTheWeebs → Clear Data

3. **Try different login method**
   - If using Google, try email/password
   - If using email, try Google OAuth

4. **Reset password**
   - Use "Forgot Password" on login screen

---

### Issue 3: Features Not Loading / Blank Screen

**Fixes:**
1. **Check storage space** - Need at least 500MB free

2. **Update the app** - Old versions may have bugs

3. **Check internet speed** - Need at least 5 Mbps

4. **Switch networks**
   - Try WiFi instead of mobile data
   - Or vice versa

5. **Allow app permissions**
   - iOS: Settings → ForTheWeebs → Enable all permissions
   - Android: Settings → Apps → ForTheWeebs → Permissions → Allow all

---

### Issue 4: Can't Upload Files

**Fixes:**
1. **Check storage limits**
   - Free tier: 100MB total
   - Check your current usage in Settings

2. **File size too large**
   - Free tier: Max 10MB per file
   - Paid tiers: Max 100MB per file

3. **Allow storage permissions**
   - iOS: Settings → ForTheWeebs → Photos → All Photos
   - Android: Settings → Apps → ForTheWeebs → Permissions → Storage

4. **Try smaller file**
   - Compress image/video first
   - Use lower resolution

---

### Issue 5: Exports Failing / Stuck

**Fixes:**
1. **Keep app open** - Don't minimize during export

2. **Check internet connection** - Exports upload to cloud

3. **Wait longer** - Large files take time
   - 1080p video: 2-5 minutes
   - 4K video: 5-15 minutes

4. **Reduce quality**
   - Try 720p instead of 1080p
   - Try PNG instead of high-quality JPG

---

### Issue 6: VIP/Paid Features Locked

**Fixes:**
1. **Log out and log back in**

2. **Check subscription status**
   - Go to Settings → Subscription
   - Verify payment went through

3. **Restore purchases** (if you paid)
   - iOS: Settings → Restore Purchases
   - Android: Settings → Sync Account

4. **Contact support** if still locked
   - Email: support@fortheweebs.com
   - Include: Email, tier purchased, receipt

---

### Issue 7: App is Slow / Laggy

**Fixes:**
1. **Close other apps** - Free up RAM

2. **Clear app cache**
   - iOS: Reinstall app
   - Android: Clear cache in settings

3. **Restart phone**

4. **Update phone OS** - Old OS may cause issues

5. **Check phone specs** - Minimum requirements:
   - 4GB RAM
   - iOS 13+ or Android 8+

---

### Issue 8: Can't Save Projects

**Fixes:**
1. **Check storage space** - Phone and app storage

2. **Check internet connection** - Projects save to cloud

3. **Try manual save**
   - Click Save button explicitly
   - Don't rely on auto-save

4. **Export project** - Download locally as backup

---

### Issue 9: Camera/Face Tracking Not Working

**Fixes:**
1. **Allow camera permissions**
   - iOS: Settings → ForTheWeebs → Camera → Allow
   - Android: Settings → Apps → ForTheWeebs → Permissions → Camera

2. **Check lighting** - Face tracking needs good light

3. **Clean camera lens**

4. **Restart app**

5. **Try front camera** - Works better for face tracking

---

### Issue 10: Can't Share to Social Media

**Fixes:**
1. **Export first** - Save to camera roll

2. **Check app permissions** - Allow access to photos

3. **Share from camera roll** - Open photo, use share button

4. **Try different format**
   - Instagram: Use PNG or JPG
   - TikTok: Use MP4
   - Twitter: Use MP4 or GIF

---

## 🛠️ Advanced Troubleshooting

### Complete App Reset (Last Resort)

**iOS:**
1. Delete ForTheWeebs app
2. Restart iPhone
3. Reinstall from App Store
4. Login again

**Android:**
1. Settings → Apps → ForTheWeebs
2. Clear Storage and Clear Cache
3. Force Stop
4. Restart phone
5. Reopen app
6. Login again

⚠️ **Warning**: This deletes all local data. Projects saved to cloud are safe.

## ✅ Prevention Tips

### Keep App Working Smoothly:

1. **Update regularly** - Check app stores weekly

2. **Don't fill storage** - Keep 500MB+ free

3. **Good internet** - Use WiFi for uploads/exports

4. **Close when done** - Don't leave app running 24/7

5. **Restart weekly** - Fresh start prevents issues

6. **Save often** - Don't lose work
